# Connect

A modern **news/blog website** built with React.js. It features multiple categories like Technology, Fashion, and more.

## ✨ Features

- Clean and responsive design
- Multiple content categories (Tech, Fashion, etc.)
- Smooth navigation with React Router
- Modern UI with sliders (React Slick)
- Fully responsive for mobile, tablet, and desktop

## 🚀 Live Demo

[View Live Site](https://Victoryruro.github.io/connect)

## 🛠️ Technologies Used

- **React.js** (Create React App)
- **React Router DOM** - Routing
- **React Slick + Slick Carousel** - Image sliders
- **CSS** - Custom styling
- **JavaScript (ES6+)**
